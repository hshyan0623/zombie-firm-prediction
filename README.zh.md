# 提前一年预测僵尸企业

**从财务报表预测企业僵尸化的机器学习流程 — 中日上市公司，2016–2024**

本项目用中日上市公司约5万条"企业-年"数据,构建一个二分类模型,预测企业**下一年**是否会变成僵尸企业。它建立在剑桥大学 MPhil 论文(政府支援与系统性风险的实证研究)的基础上,并把该研究中的僵尸识别理论(FN-CHK 法)转化为**预测特征**。

## 为什么这不只是"跑个模型"

| 设计选择 | 展示了什么 |
|---------|-----------|
| **防泄露的预测目标** | 僵尸标签是用当年财务算出的,预测当年等于泄露答案。我把标签前移一年(`t`特征 → `t+1`标签),迫使模型做真正的预测。 |
| **领域知识特征化** | 把论文里的利率缺口理论(CHK法)变成预测特征——连接计量经济与机器学习。 |
| **类别不平衡** | 正例(僵尸)约8%。用 `class_weight` / `scale_pos_weight` 处理,以 **PR-AUC** 为主评估,而非准确率。 |
| **按时间划分** | 不用随机划分,而是用过去(2016–2020)训练、未来(2021–2022)测试,贴近真实部署。 |
| **可解释性** | 用 SHAP 揭示哪些财务信号驱动预测,并检验它们是否与经济理论一致。 |
| **错误分析** | 剖析假阳性/假阴性,理解模型在哪里、为什么犯错。 |

## 核心结果

| 模型 | AUC | PR-AUC |
|------|-----|--------|
| 逻辑回归(基线) | 0.927 | 0.591 |
| **随机森林(最优)** | **0.954** | **0.672** |
| XGBoost(调优后) | 0.952 | 0.663 |
| 随机基准 | 0.500 | 0.095 |

PR-AUC 0.67 约为随机基准(0.095)的 **7 倍**——对"提前一年预测僵尸化"来说是实用的精度。

### 超越指标的发现

1. **特征胜过模型复杂度。** 精心设计、理论驱动的特征工程,让简单的逻辑回归就达到 PR-AUC 0.59,最复杂的模型也只多一点。这个问题里,领域知识特征比模型选择更重要。
2. **理论被机器学习验证。** SHAP 把资产负债率和 ROA 排在最前,利息类特征(隐含利率、**利率缺口(CHK)**、利息保障倍数)进入前六。经济学与机器学习一致。
3. **僵尸化跨越国界。** 国家(`is_china`)在 SHAP 重要度里靠后:僵尸企业的财务特征基本超越国界。
4. **模型盲区有启发。** 漏判的僵尸(假阴性)集中在"表面财务在好转、但仍依赖隐性支援"的企业,凸显了利率缺口这类"本质"特征的必要性。

## 方法概览

**目标变量。** 对每个"企业-年"`t`,标签是该企业在`t+1`年是否为僵尸(FN-CHK)。僵尸识别把利率缺口条件与盈利、杠杆标准结合,使用各国政策利率(中国:人民银行;日本:日银平均约定利率)。

**5大类18个特征。** 负债结构(杠杆、期限结构)、利息与偿债(隐含利率、**利率缺口**、保障倍数)、盈利能力(ROA、亏损标志)、动态趋势(各种增长率、连续亏损年数、ROA波动率)、国籍。全部为比率,不同规模企业可公平比较;极端值经 inf→NaN 与 1%/99% 缩尾处理。

**模型。** 基线用逻辑回归(中位数填补+标准化);主力用随机森林和 XGBoost(能原生处理缺失值)。不平衡用加权处理;超参数在**仅训练集内部**用交叉验证做网格搜索调优。

## 深度版:仅中国、特征扩展模型

第二个notebook(`notebooks/zombie_prediction_china_deep.ipynb`)构建了仅用中国数据的26特征模型,新增了营收类指标(净利润率、EBIT利润率、总资产周转率——杜邦式分解)、净利润与EBIT背离信号、成长性特征和行业标记。这些字段只有中国有(日本Osiris缺失),所以这个变体是用"跨国覆盖"换"特征深度"。

**新增特征揭示了什么:**

- **营收类盈利指标有效。** 净利润率、EBIT利润率排第5、第6——营收数据带来的精细盈利刻画有实际预测价值。
- **核心信号极其稳健。** 前四名特征(资产负债率、ROA、短债依赖、利率缺口)在完全不同的特征集下,与主模型完全一致。僵尸企业的本质就是杠杆、盈利、优惠利率。
- **两个先验假设被数据否定,而且很有启发。** 用来捕捉"虚假成长"的应收-营收增长差排在末尾——它指向财务造假,和僵尸化是不同的问题。而行业标记(高新、金融)排在**最后两名**(第26、第25,重要度≈0)。这是个有意义的发现:一家企业是不是僵尸,取决于它实际的财务动态,而非行业标签。模型无需行业标签就能处理"高新伪僵尸"的担忧——财务数据已经编码了一切。

更深的教训:财务行为本身携带信号,身份标签毫无增益。

## 数据与隐私

企业级原始数据(中国 CSMAR、日本 Osiris)属于专有数据,**已排除**。仅共享匿名化特征:企业标识是不透明编码(如 `CN_000001`),数值全是计算后的比率,不含公司名和原始财务金额。
